# Gitleaks Security Scanner

This repository implements automated secret scanning using Gitleaks.

## Security Checks
- Runs on every Pull Request to main branch
- Detects sensitive information like:
  - API keys
  - Access tokens
  - Private keys
  - Passwords
  - Certificates
